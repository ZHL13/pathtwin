# PathTwin v0.1.2

Patch release for remote root selection on very large folders.

## Fixed

- Replaced the system folder picker for Remote Root with a lightweight in-app picker.
- The remote picker lists only the current folder level instead of asking the OS dialog to materialize a huge directory.
- The picker shows folders and files for the current level, with folders navigable by double-click.
- Each displayed level is capped at 300 entries to keep the UI responsive on NAS/SMB shares.
- The existing remote directory tree also uses the 300-folder per-level cap.

## Included Artifacts

- `PathTwin-0.1.2-win-x64.exe`: versioned standalone executable.
- `PathTwin-latest-win-x64.exe`: same standalone executable under a stable latest filename.

GitHub also provides automatic source archives for the release tag. No separate source zip is uploaded.

## Notes

Use the Remote Root path box directly for very large or slow shares, or open Browse and navigate one folder level at a time.

rclone users should download `rclone.exe` from the official rclone downloads page and set the executable path in Settings: https://rclone.org/downloads/
