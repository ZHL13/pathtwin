# PathTwin v0.1.0

Initial MVP release for Windows.

## Highlights

- Avalonia/.NET 8 desktop app shell
- JSON profile configuration
- Remote/local/history/log root settings
- Remote directory tree with checkbox folder selection
- Start Work Session pull with relative paths preserved
- Session manifest saved locally
- End Work Session push with three-way planning
- Remote backup before overwrite/delete
- Basic logs and conflict/error reporting
- Single-instance guard
- Optional Windows scheduled task entry point via `--auto`
- Bundled-rclone support for release artifacts

## Included Artifact

The Windows artifact is expected to contain:

- `PathTwin.App.exe`
- Avalonia/Skia native runtime DLLs
- `tools/rclone.exe` when bundled
- `THIRD_PARTY_NOTICES.md`
- `third_party/rclone/COPYING`

## Notes

This is an MVP. Use dry-run and test folders first before relying on it for important work.
