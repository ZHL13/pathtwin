# PathTwin v0.1.0

Initial MVP release for Windows.

## Highlights

- Avalonia/.NET 8 desktop app shell
- JSON profile configuration
- Remote/local/history/log root settings
- Remote directory tree with checkbox folder selection
- Start Work Session pull with relative paths preserved
- Session manifest saved locally
- End Work Session push with three-way planning
- Remote backup before overwrite/delete
- Basic logs and conflict/error reporting
- Single-instance guard
- Optional Windows scheduled task entry point via `--auto`
- Optional rclone backend path; rclone is not bundled in the public release artifact

## Included Artifacts

The release provides two Windows x64 executable assets:

- `PathTwin-0.1.0-win-x64.exe`: versioned standalone executable.
- `PathTwin-latest-win-x64.exe`: same standalone executable under a stable latest filename.

GitHub also provides automatic source archives for the release tag. No separate source zip is uploaded.

## Notes

This is an MVP. Use dry-run and test folders first before relying on it for important work.

rclone users should download `rclone.exe` from the official rclone downloads page and set the executable path in Settings: https://rclone.org/downloads/
