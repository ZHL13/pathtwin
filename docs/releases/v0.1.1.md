# PathTwin v0.1.1

Patch release for large remote folder trees.

## Fixed

- Remote folder selection no longer tries to fully materialize huge directory levels.
- Directory tree loading now caps each level at 1000 folders and shows a non-selectable notice when the level is larger.
- Child directories are loaded only when expanded.
- The app no longer probes every listed folder to detect child folders, which avoids heavy SMB/NAS round trips.
- Selecting a parent folder remains stable after lazy-loaded children appear.

## Included Artifacts

- `PathTwin-0.1.1-win-x64.exe`: versioned standalone executable.
- `PathTwin-latest-win-x64.exe`: same standalone executable under a stable latest filename.

GitHub also provides automatic source archives for the release tag. No separate source zip is uploaded.

## Notes

If a remote root contains more than 1000 folders at the same level, choose a narrower remote root or subfolder for now. Full searchable/paged tree browsing can be added later.

rclone users should download `rclone.exe` from the official rclone downloads page and set the executable path in Settings: https://rclone.org/downloads/
